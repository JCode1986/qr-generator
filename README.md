# QuickQR

QuickQR is being developed as a fast, customizable QR code generator for the browser.

## Current Status

Project foundation.

## Planned Roadmap

These items are planned and are not implemented yet:

- Static QR generation
- QR customization
- PNG and SVG downloads
- Premium options
- Stripe payment flow
- Potential dynamic QR support later

## Stack

- Next.js
- React
- JavaScript
- Tailwind CSS

## Local Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Validation

Run linting:

```bash
npm run lint
```

Run a production build:

```bash
npm run build
```

## Branch Workflow

- `main` is production-ready.
- `dev` is the integration branch.
- Feature work may use `feature/*` branches.

Check out our [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying) for more details.
